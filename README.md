# Introduction

为了更好了解`ES6` （下一代 JS） 读一读 [阮老师《ECMAScript 6 入门》](https://github.com/ruanyf/es6tutorial)
ES6 includes the following new features:

- let vs const
- string
- array
- function
- class
- async
- promise
- module
- proxy
- set
- array-buffer

# Some thinking

- 块级绑定
- `var` 声明
- 变量提升 hoisting
- 块级声明
- let 声明
- let vs const
- const 声明对象
- 块级作用域（又称为词法作用域）
- 暂时性死区 / temporal dead zone / 简写 TDZ / 临时死区
- 循环中的块级绑定（循环中的块级作用域）
- `JavaScript` 的模块体系

- 语言层面的模块功能：ESM
- promise 的生命周期
